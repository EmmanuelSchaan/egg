    • Predicted number counts

    This page provides the galaxy number counts predicted by EGG in a large set of broad bands from the U band up to the ALMA 2mm. These counts were produced from the combination of three simulated catalogs: first, a wide (1 deg²) and deep ([4.5] < 30) field; second, a wider (20 deg²) but shallower ([4.5] < 20) field for the bright end; and third, a narrower (0.05 deg²) and deepr ([4.5] < 36) field for the faint end. The deepest catalog is used when the counts go above 300,000 galaxies per deg², while the widest catalog is used for counts below 100 per deg².

    You can download a PDF file containing both the cumulative and differential counts, by clicking here, and the tabulated data can be found below.

    Legend. Predicted counts from EGG. The solid curves are the predicted counts, while the small symbols represent various observations either from GOODS-South, CANDELS or other data from the literature (in particular GAMA). You can click the images for a larger PDF version.

    Tabulated data

    You can obtain a machine-readable table for all the above curves from the links below. The counts in each band are stored in a simply ASCII file with two columns: first is the flux (in uJy) and second is the corresponding source density in number of galaxies per deg² (or galaxies per dex and per deg², for differential counts). These counts do not include foreground stars from the Milky Way.
